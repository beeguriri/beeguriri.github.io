## 📑 디스크 할당 방법

### ⭐ 연속 할당

- 파일들이 디스크의 연속적인 주소들의 집합에 할당
- 파일 접근 시 탐색 횟수 최소화
- 전체 디스크 공간이 요구량을 만족해도, 연속된 공간이 아니면 외부 단편화
- 주기적인 압축 필요



### ⭐ 연결 할당

- 각 파일은 디스크 블록들의 리스트에 연결되는데, 디스크 블록들은 디스크 전체에 분산
- 디렉터리는 파일의 첫 번째와 마지막 블록의 포인터를 가지고 있음
- 외부 단편화가 발생하지 않음
- 직접 액세스 기능을 지원할 수 없음
- 포인터를 위한 기억 공간 필요



### ⭐ 인덱스 할당

- 모든 포인터들을 하나의 장소(인덱스 블록)으로 관리하여 직접 액세스 지원
