## 📑 블루투스

### ⭐ 블루프린팅

- 공격 장치 검색 => `SDP`(Service Discovery Protocol) 이용



### ⭐ 블루스나프

- 장비의 임의 파일에 접근
- `OPP`기능 사용 => 인증 없이 간편하게 정보 교환할 수 있음



### ⭐ 블루버그

- 블루투스 장비 간 취약한 연결 관리 악용



### ⭐ 블루재킹

- 스팸메시지 보냄
- 보안 위협은 상대적으로 적음
- 10m 범위 내에서 가능
